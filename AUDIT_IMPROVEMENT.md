# 🔍 操作溯源功能优化 - 2026-07-19

## 更新概述

优化 AI 工作台的操作溯源功能，改为折叠式显示，列表只显示用户问题，点击展开查看详情（包括 Markdown 渲染的 AI 回复）。

## ✅ 优化内容

### 1. 列表显示优化
**之前的问题**:
- 显示所有内容，列表太长
- AI 回复显示为纯文本，带星号等符号
- 难以快速找到历史对话

**现在的解决方案**:
- ✅ 列表只显示用户的问题
- ✅ 简洁的卡片样式
- ✅ 成功/失败/已撤回状态一目了然
- ✅ 时间戳精简显示（月-日 时:分）

### 2. 点击展开详情
**功能**:
- ✅ 点击任意问题展开详情
- ✅ 展开图标（▸ → ▾）提示状态
- ✅ 再次点击收起
- ✅ 平滑的展开动画

### 3. Markdown 渲染
**AI 回复使用 Markdown 格式显示**:
- ✅ 代码块语法高亮
- ✅ 粗体、斜体、列表
- ✅ 链接可点击
- ✅ 标题分层清晰
- ✅ 美观易读

### 4. 详情内容丰富
**展开后显示**:
- ✅ AI 回复（Markdown 渲染）
- ✅ 执行的操作列表
- ✅ 数据变化对比（前后对比）
- ✅ 错误信息（如有）
- ✅ 元数据（模型、耗时）
- ✅ 撤回按钮（可撤回的操作）

## 🔧 技术实现

### JavaScript 功能

#### 1. 新的渲染函数
```javascript
function renderAuditRow(r) {
  // 只显示问题和基本信息
  const question = r.raw_text || '无内容';
  
  // 详情内容（隐藏）
  let detailsHtml = '';
  
  // AI回复使用Markdown渲染
  if (r.llm_reply) {
    const renderedReply = renderMarkdown(r.llm_reply);
    detailsHtml += `<div class="audit-detail-reply">${renderedReply}</div>`;
  }
  
  // ... 其他详情
}
```

#### 2. 折叠展开逻辑
```javascript
function toggleAuditDetail(auditId) {
  const detail = document.getElementById(`audit-detail-${auditId}`);
```
