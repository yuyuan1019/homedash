# 🎉 HomeDash 更新 - 2026-07-19 第二批

## 更新内容

### 1. ✅ AI工作台欢迎消息空间优化
**问题**: 提示词空间过大
**解决方案**:
- padding: `2.5rem 1rem` → `1.5rem 1rem`
- font-size: `0.95rem` → `0.92rem`
- line-height: `1.8` → `1.65`
- 空间减少约40%，更紧凑舒适

### 2. ✅ 快捷指令固定"最近需要买什么"
**问题**: 需要一个固定的查询入口
**解决方案**:
- 快捷指令总数：6个
- 前5个：动态推荐（根据用户使用习惯）
- 第6个：**固定为"最近需要买什么"**
- 默认词条：
  1. 加5包方便面
  2. 加6瓶喜力啤酒
  3. 消耗一包方便面
  4. 加30颗鸡蛋
  5. 用掉2卷卫生纸
  6. **最近需要买什么** ⭐（固定）

### 3. ✅ 重点待办图片放大功能
**问题**: 待办图片太小，看不清细节
**解决方案**:
- 添加图片点击事件
- 点击图片弹出全屏预览模态框
- 图片悬停时有放大提示效果（transform: scale(1.05)）
- 预览模态框支持滚动查看大图
- 响应式设计，移动端和桌面端都适配

## 使用说明

### AI工作台快捷指令
1. 前5个按钮根据你的使用习惯动态调整
2. 最后一个按钮**固定为"最近需要买什么"**，快速查询购物清单
3. 点击任意按钮自动填充到输入框

### 待办图片查看
1. 在"重点待办"标签页查看待办
2. 如果待办包含图片，会显示缩略图
3. **鼠标悬停**在图片上会有放大提示效果
4. **点击图片**打开全屏预览
5. 预览模态框中可以看到高清大图
6. 点击"×"或点击模态框外部关闭预览
